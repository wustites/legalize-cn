# legalize-cn

将中华人民共和国法律法规作为 Git 仓库管理。每部法律一个 Markdown 文件，每一次修订、废止或取代对应一个真实的 Git commit。主分支只保留现行有效法律，历史版本通过分支处理，严格按七大法律部门组织。

## 已完成

### 宪法及宪法相关法

- **主分支**：[中华人民共和国宪法](宪法/中华人民共和国宪法.md)（现行有效）— 1982 年宪法及五次修正案（1988/1993/1999/2004/2018）
- **历史分支**：
  - [`共同纲领`](../共同纲领) — 1949 年《中国人民政治协商会议共同纲领》
  - [`54宪法`](../54宪法) — 1954 年宪法（已失效）
  - [`75宪法`](../75宪法) — 1975 年宪法（已失效）
  - [`78宪法`](../78宪法) — 1978 年宪法（已失效）；含 1979、1980 两次修正
